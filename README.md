# environment-and-ai-toolchain
## AI Development Environment

This project uses Cursor AI for AI-assisted development and follows clean coding conventions.

## Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/KhadijaMehmood05/environment-and-ai-toolchain.git
   cd environment-and-ai-toolchain
   ```
2. Install [Node.js LTS](https://nodejs.org/).
3. Open the project folder in [Cursor](https://cursor.com/).

See [CLAUDE.md](./CLAUDE.md) for coding conventions and project guidelines.
